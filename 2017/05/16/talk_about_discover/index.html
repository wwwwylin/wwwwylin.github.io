为什么停不下刷朋友圈

Posted on 2017-05-16 | | Visitors

为什么我们停不下刷朋友圈，即便是我们可能隐约感受到刷朋友圈其实会浪费时间？下面将分别从用户心理和平台特性两方面展开分析。

用户心理和平台特性其实是相互作用的，用户心理影响平台特性的设计，平台特性的设计又会影响或转变用户心理。

朋友圈

朋友圈主要是打造了一个用户个人生活相关信息展示的平台，其他用户包括自己可以会发布的内容进行点赞或评论，但是不能转载。

用户心理

窥探欲

用户能够在朋友圈中查看其他人发布个人生活相关信息，这些信息与用户的相关度（生活紧密度）往往很高，因为是基于个人动态的出发点进行发布的。

而多数人对其他人的生活都会有“窥探”心理，想要查看或了解他人的生活状况，再进行其它的心理活动。

同时，由于朋友圈如果是非好友是不能查看的（甚至在这个基础上还有查看权限），所以会有更多的人更愿意在朋友圈中发相对私密的事情或者是同一事情更愿意在朋友圈中发布，更增强了用户窥探心理对朋友圈的依赖。

虚荣心

用户可以将自己“想要展现”的生活模样展现在朋友圈当中，并乐于看到别人对自己展现事物的评价。

攀比心理

在窥探欲（好奇他人的生活情况）的基础上还会容易引出攀比心理，想要看别人的生活其实有一部分原因也是希望了解自己与他人相比过得怎么样，或者直白来说就是看自己过得是不是比别人好。

懒惰性

即便是发现刷朋友圈其实是浪费空闲时间的一种行为，但是已经养成了“看到红点”就习惯性点开朋友圈的习惯，安于现状总是比做出改变简单。

同时朋友圈操作的简便性也助长了刷朋友圈习惯的养成，因为只要点或者是滑动的操作就能够满足前面所说的心理。

自我欺骗

自我欺骗来自两种，一种是来自于自身不自觉营造的自我欺骗，另一种是来自于其他内容引导的自我欺骗。

由于平台内容是可以自主塑造的，所以用户可以将自己的朋友圈搭建成自己喜欢的模样，并期待他人给予相关的答复，于是对朋友圈的关注会提高，再加上现在微信的使用度及普及度都十分高的基础上，希望在朋友圈得到关注的人也更有可能得到回应。

同时，我们可以发现在朋友圈里面的内容，都是偏正能量的或者是毒鸡汤的存在。这些都能够引导我们“自我欺骗”，如“生活是美好的，当下有困难跨过去，美好的大千世界正等着你”。这些对于一部分用户来说很受用。

补充：为什么说朋友圈里面的内容是偏向正能量或者是毒鸡汤

1） 偏向正能量

  • 真正的难题很多时候是很难跟他人开口的，但是可以通过平台进行宣泄，但是又不想自己软弱的形象展示出来，但是又想表达就换一种积极的方式
  • 希望在一个公共场合给自己加油鼓劲，这样的力量相对于自己给自己说来说要高得多，同时还存在“他人也希望或者也会给自己鼓励”或者“他人觉得自己好励志”等正面的遐想

2） 毒鸡汤

毒鸡汤其实可以理解为是诙谐版的“正能量”，只是相较于正能量来说其还会点出问题的症结在哪里，但是没有办法，还是要继续“正能量”地走下去！

平台特性

消息相关性

朋友圈的消息相关是(看了红点就想点）

1） 你关注好友的信息更新了会有通知（排除你关了通知）
2） 你点赞或者是评论了他人的朋友圈之后只要有新的动态就会有通知

操作简便性

浏览朋友圈只需要两个步骤点和滑，且逻辑十分简单，而且还有按赞功能的提供，简易证明我来过或者是我同意等；发朋友圈的话也是常规的点击并添加的方式。

消息互动性

朋友圈的消息是极具有互动性的，主要通过消息的相关性来体现。同时的话，由于朋友圈里边的内容也主要是展现生活特质的能容所以信息交流的互动性也会更强烈。
