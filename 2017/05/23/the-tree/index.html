大漠的树

Posted on 2017-05-23 | | Visitors
我是大漠的一棵树

很长的一段时间里 / 我是一个人

那日 / 大漠还是昨天的大漠 / 沙尘飞舞

但我听到了那歌声 / 她飘来飘去的 / 很是可爱


她好像每天都在歌唱

啾啾啾的 / 又是日出又是日落

但她好像听不懂我说的话


没过多久 / 这里又有了另一个歌声

他的沙哑许多 / 像是藏着某种力量

那晚 / 树枝在抖动 / 撞落的树叶发出沙沙声

月亮居然笑弯了眼


梦里 / 好像有人给我挠痒痒 / 从脚到手

等我醒来 / 手就无法像往常那样伸展了

我又听到了 / 又听到了新的声音

他 / 她 / 他的她 / 她的她


这天的沙尘 / 比以往刮得还要猛烈

我想我快要倒下了吧 / 可是我没有

歌声呢 / 歌声被沙尘遮盖了 / 我听不见

啾啾啾 / 啾啾啾 / 我听见了 / 我又听见了那些歌声 / 它比往常都要激烈而急促


我哭了 / 是的 / 我感觉自己在颤抖

不知为何 / 我就知道 / 那是告别

我努力地让自己的身体保持坚韧

好送送我的友人


再见 / 我的友人




望七号楼旁雪松感

致大树
