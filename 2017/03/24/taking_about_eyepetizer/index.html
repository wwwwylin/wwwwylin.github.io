开眼 Eyepetizer

Posted on 2017-03-24 | | Visitors

每日推荐5条全球精品短视频，让你大开眼界。

基本情况

开眼是一款充满设计感的精品短视频推荐应用，主打的功能是每日推荐5条全球精选的高清视频。

其优势是


应用功能方面

1）选择视频作为获取资讯的方式，视频主打全球精选的高清短视频，使得资讯内容更具有彰显力和感染力
2）十分控制视频的质量和时长的同时，对每日推荐的5条视频提供自动缓存功能满足人们碎片化阅读习惯的同时，进一步拓宽了应用使用场景
3）视频的内容丰富多样，在原来仅靠编辑团队筛选推荐视频的基础上，引入了其他平台已经知名的视频专栏或作者
4）视频内容的推荐方式也做了进一步优化，从一开始只能到同类型专栏自主查找类型视频到每个视频下都附带了相关联的视频推荐及专栏或者作者关注入口

其劣势是


应用功能方面

  1. 引入平台用户互动的时间较晚，在开眼开设评论功能前，用户与平台输出内容的方式就只有自主投稿、意见反馈和点赞及分享
  2. 由于开设的评论功能时间较晚，所以虽然目前平台的用户量不小但是出现评论积累量不足的情况
  3. 增加了知名的视频专栏及作者虽然提供了更多优质视频资源的入口但是较平台初始来说显得有些臃肿

未来发展 / 改进方向


在保有原来推荐精品短视频的基础上

  1. 提升并引导用户观看习惯，不单单是休闲娱乐也可以是学习晋升，知名专栏就是很好的一个突破点，甚至未来可以在短视频中加入知识付费
  2. 优化对内容归纳及推荐设计，由于应用加入了更多的视频内容及更多样的相关推荐方式 ，用户获取内容方式增多的同时也容易造成难以下手的困窘，如何能够帮助用户快速锁定喜欢的内容并不会让用户反感甚至是迟迟找不到自己喜欢的反倒是觉得这个应用内容臃肿
  3. 加强用户与平台互动氛围及UGC内容，主要通过分享吸引更多的新用户加入，通过优质评论及点赞来增强用户对产品的粘结度，甚至是建立专业人士对用户的互动
  4. 探索平台变现之路，将广告穿插在优质的视频当中进行传播，用户会更容易接受并分享出去
